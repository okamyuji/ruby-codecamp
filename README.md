## 学習の進め方

1. 順番に学習することを推奨します。各セクションは前のセクションの知識を前提としています
2. 各問題のアルゴリズム概説を理解してから、コードを読んでください
3. 必ずコードを実際に実行して、動作を確認してください
4. 計算量（時間・空間）を意識して問題を解いてください
5. 複数の解法がある場合は、それぞれの長所と短所を理解してください

## 計算量の目安

| 計算量 | 例 | 性能 |
| ----- | -- | --- |
| O(1) | ハッシュテーブルのアクセス | 非常に速い |
| O(log n) | 二分探索 | 非常に速い |
| O(n) | 線形探索 | 速い |
| O(n log n) | マージソート、ヒープソート | まあまあ速い |
| O(n²) | 二重ループ | 遅い |
| O(2ⁿ) | 再帰（メモ化なし） | 非常に遅い |

## コーディング規約

このリポジトリのコードは以下の規約に従っています。

- Ruby 3.4を対象としています
- インデントは2スペースを使用します
- メソッド名はスネークケース（snake_case）を使用します
- クラス名はキャメルケース（CamelCase）を使用します
- コメントは日本語で記述し、わかりやすさを優先します
- 複数の解法がある場合は、すべて提示します

## コードの実行方法

各ファイルのコードブロックは、そのままRubyで実行できます。

```bash
# Rubyのバージョン確認
ruby --version

# コードの実行例
ruby -e "
def array_sum(arr)
  arr.sum
end

puts array_sum([1, 2, 3, 4, 5])
"
```

または、コードをファイルに保存して実行します。

```bash
# コードをファイルに保存
cat > test.rb << 'EOF'
def array_sum(arr)
  arr.sum
end

puts array_sum([1, 2, 3, 4, 5])
EOF

# ファイルを実行
ruby test.rb
```
